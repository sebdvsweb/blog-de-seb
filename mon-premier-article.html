Ce que j'ai fait pendant ces premiers cours

Découverte des langages HTML et CSS, prise en main du framework Bootstrap et de son système de grille, découverte du site codepen pour réaliser des exercices de code.

Le premier cours a été l'occasion de rencontrer les élèves de la promotion GT 1D Thimo et Macéo et d'en apprendre un peu sur eux. Je me suis présenté puis nous avons parlé de Garage 404, des cours de cette année, de leurs contenus et de ce que l'on attendrait d'eux.

Pour ce premier cours, un élève de seconde année nous a accompagné. Les deux nouveaux élèves ont présenté leurs projets Scratch réalisés par le passé, et le GT2 leur a présenté son site réalisé l'an passé sur le framework Bootstrap, que nous allons étudier cette année.

HTML et CSS à travers CodePen

Codepen.io est un site Internet nous permettant de tester rapidement tous les fragments de code qui vont nous faire découvrir le balisage HTML. Avec une interface simple, le site vous permet d’écrire du code HTML, CSS, et Javascript et de simuler la page que les codes sont censés créer.

  • Le premier langage étudié est le HTML. C’est un langage qui permet de composer des pages web. On parle de langage de balisage et non de langage de programmation, car le but du HTML est d’encadrer les différents éléments présents dans une page (images, titres, paragraphes ...) par des balises.
  • CSS (Cascading Style Sheets) est un langage de styles. Il permet d'apporter du design au contenu HTML intégré. CSS est capable de changer tous les aspects de design d'une page Web comme les couleurs, les positions, les transformations, les animations et bien beaucoup d'autres choses.

C'est quoi Bootstrap ?

Bootstrap est une collection d'outils utiles à la création du design de sites et d'applications web. C'est un ensemble qui contient des codes HTML et CSS, des formulaires, boutons, outils de navigation et autres éléments interactifs, ainsi que des extensions JavaScript en option, tout cela étant prêt à l'emploi. Bootstrap s'appuie sur un système de découpage de l'écran en grille ou grid en anglais.

“Quickly design and customize responsive mobile-first sites with Bootstrap, the world’s most popular front-end open source toolkit, featuring Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful JavaScript plugins.”

Grid System (Système de grille) est un des concepts les plus importants de Bootstrap. Il vous donne le moyen d'organiser les composants sur une interface et pour que votre website soit compatible avec différents appareils. Le système de grille (Grid System) de Bootstrap contient plusieurs lignes mais il n'y a que 12 colonnes. En détail :

  • Pour utiliser ce système de grilles Bootstrap, nous allons avant tout devoir définir un élément conteneur dans notre page HTML auquel on passera un attribut class="container" (taille fixée en pixel) ou class="container-fluid" (taille full screen). La classe .container va permettre de définir un conteneur adaptable ou « responsive » de taille fixe, ce qui signifie que notre conteneur aura toujours la même taille pour un breakpoint donné et changera de taille à chaque breakpoint.

  • Dans le système de grille de Bootstrap, les lignes ou row en anglais doivent absolument être placées dans des conteneurs et ne sont utilisées que dans le but de servir elles-mêmes de conteneurs pour les colonnes de la grille

  • Pour chaque nouvelle ligne, on va pouvoir décider du nombre de colonnes dont la ligne va disposer à partir des 12 de base grâce à des classes .col-*. On va ensuite pouvoir « grouper » certaines de ces 12 colonnes de base ensemble dans chaque ligne pour former de nouvelles colonnes personnalisées qui contiendront nos éléments HTML (texte, image, etc).

Ce blog est entièrement conçu avec le système de grille de Bootstrap.
